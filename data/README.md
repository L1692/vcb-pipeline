# Data directory

Place your exported CSV files here (git-ignored), named after the instrument:
`XAUUSD.csv`, `DAX.csv`, `NASDAQ.csv`, ...

Supported formats (auto-detected):
- TradingView "Export chart data" (comma-separated, unix or ISO timestamps)
- MetaTrader 5 bar export (tab-separated, `<DATE>`/`<TIME>` columns)
